
# SpotifyToTweet

Very basic python scrip to tweet the song you are currently listening on Spotify.


## API Reference

 - [Tweepy](https://docs.tweepy.org/en/stable/)
 - [Spotify](https://developer.spotify.com/documentation/)


## Run

Using Python3 or any IDE.

```bash
  python3 main.py
```
